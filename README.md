### This repository contains the codes to reproduce the plots in paper *Reaching a Consensus in Predictive Loops*

* yelp_sus_var.py produces Figure 2.(b), Figure 5.(b).

* pokec_sus_var.py produces Figure 4.(b), Figure 5.(a)

* yelp_mean_estimation.py produces Figure 7.(a), Figure 6.(c), 6.(d).

* pokec_mean_estimation.py produces Figure 3.(a), Figure 6.(a), 6.(b).

* yelp_retrain_steps.py produces Figure 2.(a), Figure 3.(b).

* pokec_retrain_steps.py produces Figure 4.(a), Figure 7.(b).
